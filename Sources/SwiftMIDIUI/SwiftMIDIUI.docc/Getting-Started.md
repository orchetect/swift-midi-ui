# Getting Started

A quick guide on getting started using SwiftMIDIUI.

![swift-midi-ui](swift-midi-ui-banner.png)

## 1. Import the Library

Add the SwiftMIDIUI module to your project, and import it.

```swift
import SwiftMIDIUI
```

## 2. Example Code

See the [Examples](https://github.com/orchetect/swift-midi-ui/tree/main/Examples) folder for a demonstration of the available views.
